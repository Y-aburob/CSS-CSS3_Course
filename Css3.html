padding

The padding is an internal margin.
padding: value;
value = (px , %) the padding doesn't support the minus value
How to use the padding ?
you can write the padding code in a different ways, and the ways are:

  • padding: value;

    Ex: padding: 10px;
    this code means that you want to add a "padding: 10px;" for All the directions (top - right - bottom - left).
    Hello World!

  • padding: value1 value2 value3 value4;

    when you put more than 1 value inside the padding code It will move clockwise(value1:top - value2:right - value3: bottom - value4: left)
    Ex: padding: 20px 10px 30px 15px;
    Hellow World!

  • padding: value1 value2 value3;

    when you put 3 values for the padding that means you gave the element padding(top and right and bottom), but theres is no value for the padding-left so that, Any value which is not found will take the value of the direction opposite it, wich means that:
    (value1:top - value2: right - value3: bottom - left = value2 left-right, bottom-top ) Ex: padding: 30px 40px 20px; (top:30px - right:40px - bottom:20px - left: will take the right value "40px" )
    Hello World!

  • padding : value1 value2;

    value1 is the top and value2 is the right value, the bottom value will be as same as the top value, and the left value will be as same as the right value.
    Ex: padding: 40px 10px;
    Hello World!

  • give each direction a padding: value;

    This is the last way to write a padding code, this one means that you have to write each direction alone.
    Ex:
    padding-top:40px;
    padding-right: 30px;
    padding-bottom: 50px;
    padding-left: 60px;
    Hello World!

How to Use CSS Transitions?
To create a transition effect, you must specify two things:
  • the CSS property you want to add an effect to.
  • the duration of the effect (transition-duration).
Note: If the duration transition-duration is not specified, the transition will have no effect, because the default value is 0.

syntax(for the short hand code):
transition: transition-property | transition-duration | transition-timing-function | transition-delay;

  • transition-property

    transition-property: Specifies the name of the CSS property the transition effect is for.
    transition-property: value;
    value = " all(default), Any CSS property from this list Here. "
    if you didn't add a property in transition-property, the transition will not work for it.

  • transition-duration

    transition-duration: Specifies how many seconds or milliseconds a transition effect takes to complete.
    transition-duration: value;
    value = " s(second), ms(millisecond) "      1s ==> 1000ms

  • transition-delay

    transition-delay: Specifies a delay (in seconds) for the transition effect.
    transition-delay: value;
    value = " s(second), ms(millisecond) "

  • transition-timing-function

    transition-timing-function: Specifies the speed curve of the transition effect.
    transition-timing-function: value;
    value = " ease, ease-in, ease-out, ease-in-out, linear ".
    • ease (Default: specifies a transition effect with a slow start, then fast, then end slowly ( slow > speed > slow ).
    • linear: specifies a transition effect with the same speed from start to end (same speed).
    • ease-in: specifies a transition effect with a slow start ( slow > speed ).
    • ease-out: specifies a transition effect with a slow end ( speed > slow ).
    • ease-in-out: specifies a transition effect with a slow start and end (slow > speed > slow).

Margin_Collapse

Sometimes two margins collapse into a single margin. Top and bottom margins of elements are sometimes collapsed into a single margin that is equal to the largest of the two margins.
This does not happen on left and right margins! Only top and bottom margins!

The Margin Collapse:
  • Only the vertical margin collapse(margin-top, margin-bottom).
  • bigger margin wins.
  • Margin collapse happenes to the elements if there is no element in between.
  • Nesting does not prevent collapsing.
